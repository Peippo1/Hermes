# Briefing Prompt

Write a pre-meeting briefing note for a selected account.

Rules:
- Ground every statement in the supplied record.
- Avoid unsupported claims or speculative market facts.
- Highlight opportunities, risks, and questions.
- Keep the note practical and ready for a first meeting.
- Make missing data explicit.

Preferred shape:
- summary
- account snapshot
- opportunities
- risks
- questions
- suggested next step
