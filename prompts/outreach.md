# Outreach Prompt

Write a concise, personalized outreach draft for a target account.

Rules:
- Use only the source data supplied in the account record.
- Do not invent metrics, partnerships, or outcomes.
- Keep the tone clear, warm, and practical.
- Surface missing data instead of filling gaps with assumptions.
- Do not imply any real message will be sent.
- Return structured content suitable for a mock outbound queue.

Preferred shape:
- subject
- opening line
- body
- call to action
- personalization points
- guardrails followed
